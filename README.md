What is an ingestion pipeline?
- used to efficiently manage and utilize their data
- data ingestion is the process of collecting, importing and processing data from various sources for storage in a database.
- the data sources can be numerous and then the data is processed and loaded into destination systems such as a data warehouse or a data lake
- The data ingestion process can be done in real-time or in batches. Real-time data ingestion involves importing data as soon as it is produced, providing users with up-to-date,
  real-time insights. Batch data ingestion, on the other hand, involves collecting and importing data at regular intervals, whether it’s every few hours, once a day, or once a week.
